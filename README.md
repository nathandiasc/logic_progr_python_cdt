Este repositório é dedicado aos fundamentos da Lógica de Programação, utilizando Python como linguagem principal para a implementação dos conceitos. Perfeito para iniciantes que buscam construir uma base sólida em algoritmos e estruturas de dados essenciais.

# 🐍 Lógica de Programação com Python

Este repositório foi criado com o objetivo de reunir estudos e exercícios relacionados aos **fundamentos da Lógica de Programação**, utilizando a linguagem **Python** como ferramenta principal para a implementação dos algoritmos.

Os exemplos apresentados buscam demonstrar, de forma simples e prática, como podemos transformar tarefas e situações do cotidiano em uma sequência lógica de **passos, instruções e procedimentos**, desenvolvendo o pensamento computacional e a capacidade de solucionar problemas.

## 🎯 Objetivos

* Desenvolver o raciocínio lógico e o pensamento computacional;
* Compreender o conceito de algoritmo;
* Aprender a representar problemas por meio de uma sequência de passos;
* Praticar a implementação de algoritmos utilizando Python;
* Desenvolver uma base para o estudo de programação e estruturas de dados;
* Relacionar conceitos de programação com situações presentes no cotidiano.

## 📚 Conteúdos

Neste projeto, são trabalhados conceitos introdutórios de **Lógica de Programação** por meio de algoritmos que representam procedimentos do dia a dia.

Entre os exemplos disponíveis estão:

* 🍰 Como fazer um bolo;
* 🍔 Como fazer um hambúrguer;
* 🍟 Como fritar batatas;
* 🍳 Como fritar um ovo;
* 🧴 Como lavar o cabelo;
* 🚗 Como trocar um pneu.

Esses exemplos ajudam a compreender que um algoritmo pode ser entendido como uma **sequência ordenada de instruções necessárias para realizar uma determinada tarefa ou resolver um problema**.

## 🚀 Como executar os programas

Para executar os exemplos deste repositório, é necessário ter o **Python** instalado em seu computador.

Depois de clonar ou baixar o repositório, acesse a pasta do projeto pelo terminal e execute o arquivo Python desejado.

Exemplo:

```bash
python como_fazer_bolo.py
```

Você também pode executar os demais arquivos substituindo o nome do arquivo no comando.
